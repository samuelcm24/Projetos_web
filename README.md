# Projetos Web

Repositório criado para reunir meus projetos de desenvolvimento web feitos durante meus estudos em Programação IV e práticas pessoais.

O objetivo deste repositório é organizar projetos usando **HTML**, **CSS** e **JavaScript**, registrando minha evolução no desenvolvimento front-end e criando materiais que possam ser apresentados em portfólio acadêmico e profissional.

---

## Objetivo do repositório

Este repositório serve para armazenar projetos web desenvolvidos com foco em:

- estruturação de páginas com HTML;
- estilização com CSS;
- interatividade com JavaScript;
- organização de arquivos;
- boas práticas de desenvolvimento;
- criação de páginas temáticas e interativas.

---

## Tecnologias utilizadas

Os projetos deste repositório podem utilizar:

- HTML5
- CSS3
- JavaScript
- Flexbox
- Grid Layout
- Animações com CSS
- Manipulação do DOM
- Eventos com JavaScript
- Consumo de dados com JSON/API, quando necessário
