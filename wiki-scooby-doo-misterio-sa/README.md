
# 🕵️ Wiki Scooby-Doo! Mistério S/A

Projeto desenvolvido para a disciplina de **Programação IV**, com o objetivo de criar uma **wiki interativa temática** utilizando **HTML**, **CSS** e **JavaScript**.

O tema escolhido foi **Scooby-Doo! Mistério S/A**, uma versão mais sombria e misteriosa da clássica franquia Scooby-Doo, ambientada na cidade de **Crystal Cove**, conhecida por seus mistérios, criaturas assustadoras e segredos do passado.

---

## 📌 Sobre o projeto

A proposta deste projeto é construir uma página no estilo de uma mini enciclopédia digital, reunindo informações sobre a série **Scooby-Doo! Mistério S/A**.

A wiki apresenta conteúdos como:

- visão geral da série;
- temporadas;
- personagens;
- galeria;
- mistérios;
- curiosidades;
- elementos interativos com JavaScript.

O projeto foi pensado para ter uma identidade visual inspirada no clima de mistério da série, utilizando fundo animado, cores escuras, brilho dourado, cards informativos e efeitos visuais.

---

## 🎯 Objetivos do trabalho

Este projeto foi desenvolvido com foco em praticar:

- estruturação de páginas com HTML;
- uso de tags semânticas;
- organização de conteúdo em seções;
- estilização com CSS;
- criação de layout com Flexbox;
- uso de efeitos visuais com `hover`, `transition` e animações;
- interatividade com JavaScript;
- manipulação do DOM;
- organização de arquivos para um projeto web.

## ✨ Funcionalidades

O projeto conta com:

- cabeçalho com menu de navegação;

- efeito de brilho dourado no título;

- galeria de personagens;
